Live Braille Translator Project

About the Project

Live Braille Translator adalah project yang kami buat untuk mata kuliah Proyek Telematika. Alat yang kami buat nantinya dapat memkonversi suatu kalimat tertentu dan nantinya akan di ubah kedalam bentuk huruf braille. Dan kami menggunakan solenoid yang membentuk 6 titik timbul untuk membuat huruf braille tersebut.

How It Works

- Users akan menginput text di web yang sudah tersedia.
- Lalu, text yang diinput di web akan dikirim ke arduino IDE.
- Lalu, text tersebut akan diproses.
- Dan, membuat output yang akan ditunjukan oleh gelombang solenoid yang membentuk huruf braille.

Features

  • Translate text-to-Braille secara instan/real time.
  • Clear dan user-friendly interface
  • Dapat mengubah jenis braille
